Introdução

Bem-vindo à documentação oficial do projeto Zénia! Aqui você encontrará informações detalhadas sobre o uso da IA, regras de conduta, licenciamento e integração com o plano Open Source da Netlify.

Código de Conduta

Este Código de Conduta define as normas de comportamento para todos os utilizadores, colaboradores e sistemas que interagem com o ecossistema Zénia.

1. Respeito e Boa Conduta

  • Assédio verbal ou escrito não é permitido.
  • Discriminação de qualquer tipo não é aceita.
  • Intimidação ou ataques pessoais são proibidos.

2. Segurança e Proteção

  • Hacking, exploração de bugs ou injeção de código malicioso são proibidos.
  • Relate vulnerabilidades imediatamente ao administrador.

3. Privacidade

Não deve ser coletado ou compartilhado nenhum dado sensível sem permissão.

4. Uso Responsável da IA

  • Usar a IA apenas para aprendizado, desenvolvimento e assistência prática.
  • Proibido uso para malware, golpes, conteúdo ofensivo ou atividades ilegais.

5. Comunicação Adequada

  • Spam, discurso de ódio ou conteúdo abusivo não são permitidos.

6. Inclusão

Todos são bem-vindos, o ambiente deve ser seguro e inclusivo.

7. Responsabilidade

Cada utilizador é responsável pelas suas ações; violações podem resultar em advertência, suspensão ou bloqueio.

8. Conteúdos Aceitos e Proibidos

  • Permitidos: Educação, projetos, inovação, cultura segura.
  • Proibidos: Conteúdos ilegais, exploração de menores, prejuízo a terceiros, dados privados sem consentimento.

9. Atualizações

O Código de Conduta pode ser atualizado conforme a evolução do projeto.

Licença

O projeto deve possuir uma licença Open Source válida, como MIT, Apache-2.0, GPL ou CC BY/CC0. Licenças proprietárias não são aceitas.

Link para Netlify

Inclua no site o link obrigatório:

O link deve estar na página inicial ou em todas as páginas internas.
